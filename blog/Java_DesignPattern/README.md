# 《On Java 8》中文版

## 书籍简介

* 本书原作者为 [美] Bruce Eckel，即《Java 编程思想》的作者。
* 本书是事实上的 《Java 编程思想》第五版。
* 《Java 编程思想》第四版基于 JAVA **5** 版本；《On Java 8》 基于 JAVA **8** 版本。
